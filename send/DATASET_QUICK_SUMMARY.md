## 使用方法

### 完整训练 (~35分钟)

```bash
cd /root/agent/send
python3 sender.py --simulate --interval 30 --config-file network_config.csv
```

### 快速测试 (前20个配置，~10分钟)

```bash
head -21 network_config.csv > network_config_quick.csv
python3 sender.py --simulate --interval 30 --config-file network_config_quick.csv
```

### 加速训练 (15秒间隔，~18分钟)

```bash
python3 sender.py --simulate --interval 15 --config-file network_config.csv
```
